¿Qué es y para qué sirve este proyecto?

Es una aplicación de preguntas y respuestas de caracter general. Una vez terminado el juego, muestra los resultados de todas las preguntas. Para hacerla se utilizó TypeScript, Angular y Bootstrap como framework de diseño.
