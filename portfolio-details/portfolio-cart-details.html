¿Qué es y para qué sirve este proyecto?

Es una plantilla de lo que podría ser una tienda con un carrito de compras incluido, mostrando el funcionamiento del mismo. Para hacerlo se utilizó vanilla JavaScript y Skeleton como librería de diseño.
